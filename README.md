# TIKA SAATHI — PWA Setup

## Deploy to GitHub Pages (FREE)
1. Create repository `tika-saathi` on GitHub
2. Upload ALL files (index.html, manifest.json, sw.js, icons folder)
3. Settings → Pages → Source: main → Save
4. Live at: https://yourusername.github.io/tika-saathi/

## Install on Phone
- Android: Visit the URL → tap "Install App" banner
- iPhone: Safari → Share → "Add to Home Screen"
